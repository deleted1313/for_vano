# тупой калькулятор

## Project setup(установка)
```
yarn install
или
npm i
```

### Compiles and hot-reloads for development(разработка)
```
yarn run serve
или
npm run serve
```

### Compiles and minifies for production(билд для прода)
```
yarn run build
или
npm run build
```

### Lints and fixes files
```
yarn run lint
```

### Run your unit tests
```
yarn run test:unit
```
